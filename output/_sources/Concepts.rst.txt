Concepts
========================

Dataframe
-------------------------

Una colección distribuida de datos agrupados en columnas nombradas

Null Values
-------------------------

* Usar la función Dataframe.fillna para reemplazar todos los valores nulos de un dataframe

Pandas UDF vs Spark UDF
--------------------------

* La diferencia principal entre pandas UDF y Spark UDF es este último procesa los datos fila a fila mientras que pandas UDF lo hace bloque por bloque.
