Operaciones con fechas
=============================

Fecha y Hora actual
------------------------

.. py:function:: datetime.now()

Ejemplos:

.. code-block:: python

    from datetime import datetime
    newdt = datetime.now()

Fecha actual
--------------

.. py:function:: date.today()

Ejemplos:

.. code-block:: python

    from datetime import date
    newdate = date.today()
