DateTime
==================

.. py:function:: datetime.strptime(value, format)

    Parsea/convierte un string a un objeto *datetime*.

    Parámetros:
        - *value*: Un string representando la fecha.
        - *format*: El formato en el que es informado value.

    1. Retorna un objeto *datetime*
    2. Si el formato no es un formato válido devuelve la excepcion *ValueError*
