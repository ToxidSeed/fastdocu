DataFrame.pivot
==============================

pivota una columna del actual dataFrame y realiza la agregaci√≥n indicada. Hay dos tipos de versiones de pivot: Uno que requiere el caller para especificar la lista de distintos valores para pivotar, y uno que no.

.. py:function:: GroupedData.pivot(pivot_col, values=None)

Ejemplos:

Dataframe base para los ejemplos:
https://www.kaggle.com/anuvagoyal/sales-store-product-details

.. code-block:: python

    df = spark.read.csv("datasets/Salesstore.csv", quote='"', escape='"', sep=",", header=True)

Realizando el pivot

.. code-block:: python

    df.groupBy("Ship_Mode").pivot()
